Ouranos
=========

Importation tests for Regional Climatic Models (HDF5 format, version >7.4, ```*.mat```) outputs into PostGIS raster objects.

Dependencies
============

### Libraries 

**Python (2.7+):**

	import h5py
	import numpy
	import pandas

**R (3.0+):**

	require(sp)
	require(ggplot2)
	require(ggmap)
	require(stringr)
